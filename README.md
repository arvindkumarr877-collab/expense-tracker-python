# Expense Tracker

A simple command-line Expense Tracker built in Python.

## Features

- Add expenses
- View all expenses
- Show total expenses
- Show category-wise expenses
- Input validation
- Exception handling
- Modular code using functions

## Concepts Used

- Functions
- Lists
- Dictionaries
- Loops
- Conditionals
- Exception Handling
- Git & GitHub

## How to Run

```bash
python expense_tracker.py
```
